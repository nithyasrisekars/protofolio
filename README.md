# Personal Portfolio Website

A modern, accessible, and responsive personal portfolio website built using HTML5 and CSS3 only. This project demonstrates semantic structure, strong WCAG 2.1 accessibility practices, and search engine friendly markup.

## Project Overview

This portfolio website includes four main pages:
- `index.html` ‚Äî Home page with hero, skills, featured projects, and calls to action.
- `about.html` ‚Äî Personal biography, education, certifications, skills, and career timeline.
- `projects.html` ‚Äî Responsive project showcase with accessible cards and external links.
- `contact.html` ‚Äî Accessible contact form with required fields and validation-friendly markup.

## Accessibility Features

- Semantic HTML5 elements: `header`, `nav`, `main`, `section`, `article`, `aside`, `footer`
- Skip link for keyboard users
- Visible focus indicators for links and buttons
- Meaningful `alt` text for images
- Proper `label` associations for form fields
- `aria-label`, `aria-required`, and landmark roles where appropriate
- Keyboard-navigable interface and responsive layout
- Sufficient color contrast and readable typography

## SEO Features

- `title`, `meta charset`, `viewport`, `meta description`, `meta keywords`, `author`
- Open Graph and Twitter Card metadata
- Structured headings and clear content organization
- Favicon placeholder included

## Technologies Used

- HTML5
- CSS3

## Run Locally

1. Open the `portfolio` folder in your preferred code editor.
2. Open any HTML page in a browser, such as `index.html`.
3. No build step or server is required ‚Äî the website is static.
